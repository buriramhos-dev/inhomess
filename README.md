# ระบบ INHOMESSS - ระบบจัดการการเยี่ยมบ้านและการประเมินผู้ป่วย

## คุณสมบัติ

- ✅ ระบบจัดการข้อมูลผู้ป่วย
- ✅ การบันทึกการเยี่ยมบ้าน
- ✅ การประเมิน INHOMESSS แบบครบถ้วน
- ✅ ฐานข้อมูล MySQL อัตโนมัติ
- ✅ ดีไซน์สวยงามด้วยธีม Teal/Indigo

## การติดตั้ง

### 1. ตั้งค่าฐานข้อมูล

ระบบจะสร้างฐานข้อมูลและตารางอัตโนมัติเมื่อมีการเชื่อมต่อครั้งแรก

หรือคุณสามารถรันไฟล์ SQL ด้วยตนเอง:

```bash
mysql -u root -p < database.sql
```

### 2. ตั้งค่าการเชื่อมต่อฐานข้อมูล

แก้ไขไฟล์ `db.php` หรือตั้งค่า Environment Variables:

```php
$DB_HOST = 'localhost';
$DB_USER = 'root';
$DB_PASS = '';
$DB_NAME = 'doctorvisit';
```

### 3. เริ่มใช้งาน

เปิดเว็บเบราว์เซอร์และไปที่:
```
http://localhost/doctorvisit/
```

## โครงสร้างฐานข้อมูล

### ตาราง `patients`
เก็บข้อมูลผู้ป่วยทั้งหมด

### ตาราง `visits`
เก็บข้อมูลการเยี่ยมบ้าน

### ตาราง `inhomess_assessments`
เก็บข้อมูลการประเมิน INHOMESSS

## ไฟล์สำคัญ

- `index.php` - หน้าหลักแสดงรายชื่อผู้ป่วย
- `add.php` - เพิ่มข้อมูลผู้ป่วย
- `visit_types.php` - บันทึกประเภทการเยี่ยมบ้าน
- `inhomess_assessment.php` - ฟอร์มการประเมิน INHOMESSS
- `db.php` - การเชื่อมต่อฐานข้อมูลและฟังก์ชันช่วยเหลือ
- `styles.css` - สไตล์ธีม Teal/Indigo

## หมายเหตุ

- ระบบจะ fallback ไปใช้ JSON file ถ้าฐานข้อมูลไม่พร้อมใช้งาน
- ฐานข้อมูลจะถูกสร้างอัตโนมัติเมื่อมีการเชื่อมต่อครั้งแรก
